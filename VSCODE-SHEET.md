# Deutsche Keyboard Shortcuts VS Code

## BASIC EDITING

Multiline Cursor:

```shell
ALT + SHIFT + Zeilen selektieren
```

Zeilen auskommentieren:

```shell
STRG + #
```

Zeilen verschieben:

```shell
ALT + HOCH
ALT + RUNTER
```

Zeilen duplizieren:

```shell
STRG + ALT + HOCH
STRG + ALT + RUNTER
```
